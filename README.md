# MidBattleJump
MidBattleJump is a Starsector mod that adds jump-based hullmods for tactical deployment and retreat behavior.

## Features

- Tactical Jump Drive: ships enter battle through a short-range tactical jump.
- Tactical Retreat Jump Drive: ships can retreat through a jump after using the retreat command.
- Includes visual jump effects and size-based behavior.

## Installation

1. Download the latest release.
2. Extract the folder into your Starsector `mods` directory.
3. Enable the mod in the Starsector launcher.
